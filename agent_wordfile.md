## Flynn Agent Overview\n\nFlynn is a local-first personal OS agent built with Go and Tauri. It operates through a modular architecture:\n\n### Core Components\n- **Agents**: Handle specific tasks and workflows\n- **Models**: Local AI models for processing\n- **Memory**: Stores user preferences and facts\n- **Tools**: File operations, code analysis, web research, system control\n\n### Architecture\n- CLI interface via `cmd/flynn/`\n- Desktop app via Tauri in `desktop/`\n- Modular design with `internal/` for private code\n- Public libraries in `pkg/`\n\n### Key Features\n- Local-first processing (no cloud dependency)\n- File system operations\n- Code analysis and testing\n- Web research capabilities\n- Task management\n- System integration\n\n### Development\n- Built with Go standard toolchain\n- Uses make commands for building/testing\n- Modular package structure\n- Comprehensive testing framework
